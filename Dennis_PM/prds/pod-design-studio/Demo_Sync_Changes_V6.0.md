## 修改背景

PRD V6.0 下午进行了以下重大调整，Demo 尚未同步：

1. **底板管理替代印刷区域配置**（Print Profile 已废弃）
2. **设计方案独立管理**（不生成SKU，可被POD产品引用）
3. **创建POD产品双入口**（使用现有设计方案 / 新建设计方案）
4. **设计师改为图案上传模式**（在Admin后台按品类上传独立图案）
5. **产品组概念已废弃**（`pod_product_groups` / `pod_product_group_items` 不再使用）
