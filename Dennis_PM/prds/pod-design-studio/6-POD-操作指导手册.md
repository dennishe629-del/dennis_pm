## 模块一：Admin 印刷配置模板管理

**角色：** ShipSage Admin
**入口：** ShipSage Admin 后台 → POD → Print Profiles
**关联 Task：** T3

> **重要变更：** 印刷配置模板按**客户（company）**维护。Admin 需先选择客户，再为该客户创建/编辑印刷配置模板。不同客户的模板互不可见。

### 操作步骤

#### 1.1 新建印刷配置模板

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 进入 Admin 后台 → POD → Print Profiles | |
| 2 | **在客户筛选下拉框中选择目标客户** | 例：选择 "Company A - ABC Trading Co." |
| 3 | 点击 **[+ New Profile]** 按钮 | 所属客户已自动填入，不可更改 |
| 4 | 填写基本信息：名称、品牌、型号 | 例：名称="Bella Canvas 3001 Crew Neck"，品牌="Bella Canvas"，型号="3001" |
| 5 | 选择品类（使用 ShipSage 现有品类） | 从品类下拉框中选择，如 "T-Shirt" |
| 6 | 勾选支持的印刷面 | 正面/背面/左袖/右袖 |
| 7 | 配置各尺寸的印刷区域：为每个尺寸填写各面的宽×高（cm） | 参考实际空白品的可印刷区域测量值 |
| 8 | 添加 Mockup 底图颜色：点击 [+ Add Color]，输入颜色名称和色值 | 例：White/#FFFFFF、Black/#000000 |
| 9 | 为每个颜色上传底衫照片（正面/背面/左侧/右侧各一张） | **要求：白色背景、产品居中、高清（≥2000px）** |
| 10 | 填写热压参数默认值（温度/时间/压力） | 仓库操作时可微调 |
| 11 | 点击 **[Save]** | |

> **注意：** 同一客户同一品类下可以创建多个模板（如该客户使用了两种品牌的 T 恤空白品）。

#### 1.2 编辑印刷配置模板

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 先选择目标客户，在该客户的模板列表中找到目标模板，点击 **[Edit]** | |
| 2 | 修改需要变更的字段（名称/颜色/照片/印刷区域） | |
| 3 | 点击 **[Save Template]** | ⚠ 修改印刷区域会影响已有设计方案的印刷文件，系统会自动标记旧文件过期并重新生成 |

#### 1.3 注意事项

- 底衫照片质量直接影响效果图质量，建议使用专业产品摄影
- 印刷区域尺寸必须与实际底衫的物理可印刷区域一致，否则印刷会偏移
- 新增颜色后，需通知卖家在 SKU 映射中补充新颜色的映射

## 模块四：卖家 POD 首次引导

**角色：** 卖家（Seller）
**入口：** ShipSage OMS → Products → [+] → POD Design Studio → [+ 新建设计] → 产品关联向导
**关联 Task：** T3b

### 前置条件

- Admin 已创建该产品对应的印刷配置模板
- 卖家已在 ShipSage 中创建了各颜色/尺寸的产品 SKU（GT_Catalog）

### 操作步骤

#### 4.1 手动映射（逐条）

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 进入产品详情页 → 切换到 **POD Design** Tab | |
| 2 | 点击 **SKU Mapping** 子页面 | 展示所有颜色×尺寸的映射状态 |
| 3 | 找到状态为"⚠ 未映射"的行，点击 **[映射]** | |
| 4 | 在弹窗搜索框中输入 Customer SKU 或产品名称 | 例：输入"CT-WHT-2XL" |
| 5 | 从搜索结果中选择正确的产品 | |
| 6 | 点击 **[确认映射]** | 映射完成，状态变为"✅ 已映射" |

#### 4.2 自动匹配

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 在 SKU Mapping 页面点击 **[自动匹配]** | |
| 2 | 系统根据颜色/尺寸关键词自动搜索并显示匹配建议 | |
| 3 | 检查每条匹配建议是否正确 | 勾选正确的，取消错误的 |
| 4 | 点击 **[确认并保存]** | 批量保存选中的映射 |

#### 4.3 CSV 批量映射

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 点击 **[批量映射（CSV 上传）]** | |
| 2 | 下载 CSV 模板 | |
| 3 | 填写 CSV：每行一个 color, size, customer_sku | |
| 4 | 上传 CSV 文件 | |
| 5 | 系统校验并显示匹配结果（成功/失败行） | |
| 6 | 确认后保存 | |

## 模块五：卖家 POD 设计器

**角色：** 卖家（Seller）
**入口：** ShipSage OMS → Products → [+] → POD Design Studio → [+ 新建设计] / [编辑]
**关联 Task：** T1/T2

### 前置条件

- 印刷配置模板已由 Admin 预设
- SKU 映射已完成（至少部分颜色/尺寸已映射）

### 操作步骤

#### 5.1 进入设计器

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 进入产品详情页 → 切换到 **POD Design** Tab | |
| 2 | 点击 **[+ New Design]** | |
| 3 | 选择印刷配置模板（如果产品已关联底衫则自动选中） | |
| 4 | 选择底衫颜色（预览区显示对应颜色底衫） | |
| 5 | 进入多面设计器 | 默认加载正面 Canvas |

#### 5.2 设计操作

| Step | 操作 | 说明 |
|------|------|------|
| 1 | **切换印刷面**：点击顶部面切换按钮（正面/背面/左袖/右袖） | 各面独立编辑，互不影响 |
| 2 | **上传图片**：点击工具栏「上传图片」，选择本地文件 | 支持 PNG/JPG/SVG，≤20MB。上传后查看属性面板中的 **DPI 指示器**：绿色(≥150)=推荐，黄色(100-149)=警告，红色(<100)=建议更换 |
| 3 | **拖拽/缩放/旋转**：鼠标拖拽移动位置；拖拽角点缩放；旋转手柄旋转 | 缩放时注意 DPI 指示器变化 |
| 4 | **添加文字**：点击「添加文字」，输入内容 | 可选择字体（20+ Google Fonts）、颜色、大小 |
| 5 | **图层管理**：右键元素 → 置顶/置底/上移/下移 | |
| 6 | **撤销/重做**：Ctrl+Z / Ctrl+Y | 每面独立历史栈，最多 30 步 |
| 7 | **预览**：点击顶部 [预览] 按钮查看各面 Mockup 合成效果 | |
| 8 | 重复以上步骤为其他面添加设计（如需要） | 左侧面板显示各面使用状态 |

### 设计技巧

- **Logo 建议使用矢量 SVG 或高分辨率 PNG（≥2000px 宽）**，确保 DPI 充足
- **文字尽量使用较粗的字体**，细线字体在 DTF 印刷中可能不清晰
- **深色底衫上使用浅色设计时**，RIP 会自动加白墨底层，效果正常
- **设计区域以虚线标记**，超出虚线的部分不会被印刷

## 模块六：卖家设计方案管理

**角色：** 卖家（Seller）
**入口：** ShipSage OMS → Products → [+] → POD Design Studio → 设计方案列表
**关联 Task：** T4

### 操作步骤

#### 6.1 保存设计方案

| Step | 操作 | 说明 |
|------|------|------|
| 1 | 设计完成后点击顶部 **[保存]** | |
| 2 | 输入方案名称 | 例："Summer Logo Edition" |
| 3 | 勾选适用颜色（可多选） | 例：勾选 White + Gray |
| 4 | 决定是否设为默认方案 | 默认方案用于订单自动匹配。每个颜色只能有一个默认 |

| 操作 | 说明 |
|------|------|
| **编辑** | 点击方案的 [编辑]，重新进入设计器修改后保存。已有订单不受影响（版本隔离） |
| **设为默认** | 点击方案的默认标记，切换默认状态。同一颜色下旧默认自动取消 |
| **归档** | 不再使用的方案可归档，归档后不参与订单匹配 |

## 模块八：仓库膜打印操作

**角色：** 仓库膜打印操作员
**入口：** ShipSage WMS → POD → Print Batches
**关联 Task：** T8 Step 1

> **V1.1 变更说明：** MVP 阶段使用 Maintop RIP 内置拼版功能。ShipSage 系统提供各订单独立 PNG，操作员下载 ZIP 包后在 Maintop 中手动排布拼版，无需等待系统自动拼版。打印完成后需为每个图案打印并粘贴膜条码标签，用于后续膜预拣扫码。

### 前置条件

- DTF 打印机已开机预热
- Maintop RIP 已打开，ICC Profile 已加载
- PET 转印膜已安装到打印机
- 条码标签打印机就绪

### 操作步骤

| Step | 操作 | 系统操作 | 说明 |
|------|------|----------|------|
| 1 | 登录 WMS → POD → Print Batches | | 查看待打印批次列表 |
| 2 | 找到状态为"待打印"的批次，点击查看详情 | | 查看批次内订单列表和各订单印刷面 |
| 3 | 确认无误后点击 **[认领]** | 状态 → claimed，批次锁定到当前操作员 | |
| 4 | 点击 **[下载 ZIP 包]** | 下载批次内所有订单的独立 PNG 文件 | ZIP 内每个订单一个 PNG，含订单号+尺寸标注 |
| 5 | 解压 ZIP，打开 **Maintop RIP** 软件 | | |
| 6 | 在 Maintop 中**批量导入**所有 PNG 文件 | | File → Import All，或全选拖入工作区 |
| 7 | 在 Maintop 中**手动排布**各图案到膜幅（参考膜幅宽度 60cm） | | 尽量紧凑排列，提高膜材利用率；相邻图案保留 5mm 间距 |
| 8 | 检查 Maintop 自动生成的白墨底层预览 | | 通常无需调整；深色底衫设计注意白墨覆盖度 |
| 9 | 在 Maintop 中点击**打印** | | 打印机开始在 PET 膜上打印 |
| 10 | 打印完成后，将膜送过**撒粉机** | | 均匀撒上热熔胶粉 |
| 11 | 将膜送入**固化烘干机** | | 160°C，约 2-3 分钟 |
| 12 | 固化完成后，在 WMS 批次详情页点击 **[打印膜条码标签]** | 系统生成本批次每个图案对应的条码标签（每图案一张） | 条码与 pod_film_task 绑定 |
| 13 | 用条码打印机打印所有标签 | | |
| 14 | 将**条码标签逐一粘贴**到拼版膜上对应图案的空白边框处 | | 粘贴时核对订单号与标签一致 |
| 15 | 将膜放入**膜暂存架** | | 按批次号分格存放 |
| 16 | 回到 WMS，点击 **[标记膜就绪]** | 批次状态 → film_ready；各 pod_film_task 状态 → film_ready，可开始膜预拣 | |

>

## 模块九：仓库热压转印操作

**角色：** 仓库热压转印操作员
**入口：** ShipSage WMS → POD → Heat Press Queue
**关联 Task：** T8 Step 5-6

> **V1.1 变更说明：** Tote 拣货流程中，衣服和膜在膜预拣+衣物拣货阶段已预先放入同一 Tote Bin。热压操作员直接从 Bin 中取出衣服+膜，无需再去膜暂存架查找或从拣货区取衣服。系统在热压面板上显示效果图，操作员参考效果图确认热压位置。

### 前置条件

- 热压机已预热至目标温度（170°C）
- Tote Pick Run 膜预拣和衣物拣货均已完成（Bin 中衣服+膜已匹配）
- 系统热压任务状态为"待热压"

### 操作步骤

| Step | 操作 | 系统操作 | 说明 |
|------|------|----------|------|
| 1 | 登录 WMS → POD → Heat Press Queue | | 查看待热压任务列表 |
| 2 | 找到状态为"待热压"的任务，点击 **[认领]** | 状态 → heat_pressing | |
| 3 | 查看任务详情：订单号、Bin 编号、尺寸、颜色、需印刷的面 | | 记住对应的 Bin 编号 |
| 4 | **参考屏幕效果图**确认热压位置（居中/偏上/特殊位置等）和图案内容 | 系统显示 Warehouse Mockup 效果图 | 效果图是最终成品预览，是定位的重要参考 |
| 5 | 从 Tote Cart 上取出对应 **Bin 编号**的 Bin | | Bin 中衣服+膜已预匹配，直接取用 |
| 6 | 取出 Bin 中的**空白品（衣服）** | | 核对尺寸/颜色与任务一致 |
| 7 | 取出 Bin 中对应面的**转印膜**（膜上贴有条码标签） | | 多面订单 Bin 中会有多张膜，按印刷面顺序取用 |
| 8 | 将衣物平铺在热压机上，使用**定位板**对齐印刷区域 | | 领口对准基准点；参考效果图确认位置 |
| 9 | 将图案膜**面朝下**放在印刷区域上，对准定位板中心标记 | | |

| 11 | 打开热压机，取出衣物+膜 | | |
| 12 | 等待冷却 **5-10 秒** | | 不要立即揭膜 |
| 13 | **缓慢揭去 PET 膜** | | 从一角开始，匀速揭起 |
