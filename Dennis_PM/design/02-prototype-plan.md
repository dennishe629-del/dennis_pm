# 原型规划

> 适用范围：PRD → Demo 页面输出优先级指引。

## 建议输出顺序

| 优先级 | 页面 | 说明 |
|--------|------|------|
| P0 | 登录页 | 基础入口 |
| P0 | 工作台 / Dashboard | 首页概览 |
| P1 | 订单列表页 | 核心业务 |
| P1 | 订单详情页 | 核心业务 |
| P2 | 入库作业页 | WMS 核心 |
| P2 | 库存查询页 | WMS 核心 |
| P2 | 出库作业页 | WMS 核心 |
| P2 | 财务账单页 | 财务核心 |
| P3 | 用户角色管理页 | 系统管理 |

## 页面说明模板

| 页面名称 | 目标用户 | 页面目标 | 关键操作 | 异常处理 |
|---------|---------|---------|---------|---------|
| 示例：订单列表页 | 运营 | 查看和处理订单 | 筛选、导出、审核 | 异常订单提醒 |

## Demo 输出位置

```
prds/[模块名]/demo/
├── index.html       # 主入口/列表页
├── detail.html      # 详情页
├── [role].html      # 按角色分页
└── studio.html      # 设计器页面（如有）
```
