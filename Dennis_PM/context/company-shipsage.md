- 公司名称：**仓盛海外仓**

- 定位：**北美一站式跨境电商仓配一体化服务商**

- 仓库布局：美西、美东、美中、美南、美东南等**六大自营仓**

## 2. 核心业务介绍

### 2.1 一件代发（主营核心）
- 支持平台：Amazon、Walmart、TikTok Shop、Temu、SHEIN、独立站等
- 履约能力：当日发货率 99.9%，发货准确率 99%+
- 支持品类：小件、大件（70磅内）、家具家居、户外、家电、运动用品等
- 尾程渠道：UPS、FedEx 等官方合作
- 核心价值：多仓就近发货、时效快、尾程成本低

### 2.2 FBA 相关服务
- FBA 补货中转
- FBA 退仓换标、重贴、重新上架
- 退货质检、翻新、销毁
- 亚马逊 SPN 认证服务商、TikTok Shop 官方对接仓

### 2.3 仓储与库内运营
- 头程提柜、拆柜、打托、入库
- 收货、质检、上架、库位管理
- 库存盘点、移库、调拨、库龄管理
- 多货主库存隔离、批次/效期管理

### 2.4 多渠道一盘货履约
- 一次入库，多平台同时销售
- 系统自动分仓、自动路由最优渠道
- 库存统一、订单统一、履约统一

### 2.5 增值服务
- 退货处理、质检、分级、重上架/报废
- 换包装、组合套装、贴标、拍照核验
- 定制化仓配方案、大客户专属对接

## 4. 系统架构（PRD 生成依据）
- **WMS**：入库管理、库内作业、出库管理、库存管理、盘点、LPN/箱码、波次拣货
- **OMS**：订单接入、校验、合单/拆单、拦截、渠道匹配、面单生成、物流跟踪
- **计费系统**：仓租、操作费、运费、增值费、库龄费、自动账单
- **商家后台**：库存查询、订单跟踪、物流轨迹、报表、对账

## 5. 核心业务规则（AI 必须遵守）
1. **多货主隔离**：货主ID + 仓ID + 渠道ID 三维隔离
2. **库存状态严格分离**：可售、锁定、待检、在途、残次、异常
3. **标准入库流程**：ASN 预约 → 收货 → 质检 → 上架
4. **标准出库流程**：订单拉取 → 校验 → 波次 → 拣货 → 复核 → 打包 → 称重 → 发运
5. **异常闭环**：少货、多货、错发、串发、拦截、取消、退回、重发
6. **作业要求**：可追溯、可日志、可审计、支持并发与幂等
7. **权限体系**：商家、客服、仓管、操作员、运营、管理员分级权限
