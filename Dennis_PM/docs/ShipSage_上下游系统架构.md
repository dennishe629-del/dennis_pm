## 六、关键集成点

### 6.1 订单流集成

```
电商平台 → ShipSage OMS → DB/ERP (ASN/Order) → DB/WMS (执行)
                ↓
           DB/APP (计费)
                ↓
           DB/BI (分析)
```

### 6.2 库存流集成

```
DB/WMS (实际库存) → ShipSage → 电商平台 (库存同步)
       ↓
  DB/ERP (库存记录)
       ↓
  DB/BI (库存分析)
```

### 6.3 费用流集成

```
DB/APP (费用记录) → ShipSage Billing → 客户账单
       ↓
  DB/BI (毛利分析)
```
